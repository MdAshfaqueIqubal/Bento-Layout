# 🎨 CSS Bento Layout Project

A modern and responsive **Bento Grid Layout** made using pure HTML and CSS.  
This project focuses on learning and practicing:

- CSS Grid
- Responsive Design
- Modern UI Layouts
- Image Positioning
- Flexbox Utilities
- Clean Web Design
- 
- ## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox
- ## ✨ Features

✅ Modern Bento Style UI  
✅ Fully Responsive Layout  
✅ Clean Design  
✅ Grid-Based Structure  
✅ Rounded Cards  
✅ Image Overlay Effects  
✅ Hover Effects  
✅ Beginner Friendly Code
